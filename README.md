# AlipayShangJin

微信跳转到支付宝APP自动领取赏金红包脚本。<br/>

**iOS：** 戳链接打开页面询问你是否要跳转到支付宝APP领取红包，确定后就会自动跳转到支付宝APP并自动领取红包。<br/><br/>
**Android：** 戳链接打开页面，提醒你复制红包码并提示你前往支付宝APP搜索红包码领取，Android系统的设备无法在微信跳转到支付宝APP，所以仅起到引导作用，免去扫码过程。

